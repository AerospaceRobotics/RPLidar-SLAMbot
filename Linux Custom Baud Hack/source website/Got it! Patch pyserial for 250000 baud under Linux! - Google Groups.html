I found a patch for pyserial to set any special baudrates to ttyACMx devices such as MEGA 2560  under linux.
If you use Printrun or other python application it's for you.
I applied patch manually. For some reason patch -p2 < pyserial.patch  did  nothing.
1 download a patch from http://sourceforge.net/tracker/index.php?func=detail&aid=3607257&group_id=46487&atid=446304
2 cd /usr/lib/python2.7/site-packages/serial
3 make a copy of serialposix.py for backup
4 edit serialposix.py using patch
5 set 250000 in you marlin, compile and upload to mega.
that's all
I deleted serialposix.pyc. I think it must be recompiled on the fly.

Thanks for pointing this patch out, I've just upgraded my baudrate now.

I think the better way to install is:

svn co http://pyserial.svn.sourceforge.net/svnroot/pyserial/trunk pyserial
cd pyserial
patch -p0 < ../pyserial.patch
cd pyserial
python setup.py install --user
python3 setup.py install --user

So the next distro upgrade won't override your local version.
